# insta_cable

A Flutter Video Sharing Application.

## Live Testing 
1. Try the live demo. 
2. Watch a sample. 

## Dependencies

- `get`
- `video_player`
- `firebase_auth`
- `cloud_firestore`
- `firebase_storage` 


## Local Development

  ##### Clone the repository
```bash
git clone https://github.com/kashvigarg/insta_cable_getx.git
```
  ##### Move to the root folder
```bash
cd \insta_cable
```

  ##### To run the app, simply write
```bash
flutter run
```
